# primer2 — Documentation

[![Deploy to GitHub Pages](https://github.com/bagustris/primer2/actions/workflows/jekyll.yml/badge.svg)](https://github.com/bagustris/primer2/actions/workflows/jekyll.yml)

Documentation site for [primer2-theme](https://github.com/bagustris/primer2-theme) — a two-column GitBook-style Jekyll theme for GitHub Pages. Use [primer2-template](https://github.com/bagustris/primer2-template) to create your own page.

**[→ View the docs](https://bagustris.github.io/primer2/)**

## Docs contents

- **Introduction** — overview and features
- **Installation** — quick start via template or manual setup
- **Configuration** — `_config.yml` options and page front matter
- **Customization** — custom CSS, layout overrides, sidebar tweaks
- **Contributing** — development setup and contribution guide 
- **Markdown** -- basic mardown writing 
- **Mathajx** -- how to write math equation 
- **Mermaid** -- how to make flowchart
