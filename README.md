# primer2 — Demo Site

[![Deploy to GitHub Pages](https://github.com/bagustris/primer2/actions/workflows/jekyll.yml/badge.svg)](https://github.com/bagustris/primer2/actions/workflows/jekyll.yml)

This is the demo site for [primer2-theme](https://github.com/bagustris/primer2-theme) — a two-column GitBook-style Jekyll theme for GitHub Pages based on GitHub's [Primer](https://primer.style/) design system.

**[→ View the live demo](https://bagustris.github.io/primer2/)**

![Thumbnail of Primer](thumbnail.png)

## Related repositories

| Repository | Purpose |
|---|---|
| [primer2-theme](https://github.com/bagustris/primer2-theme) | The theme itself (`_layouts`, `_sass`, `_includes`) |
| [primer2-template](https://github.com/bagustris/primer2-template) | Starter template — click "Use this template" to create a new site |
| **primer2** (this repo) | Demo content using the theme |

## Local preview

```sh
bundle install
bundle exec jekyll serve
```

Then visit [localhost:4000](http://localhost:4000).
